To run this sample successfully you need a client certificate that provides PKI credentials.

1. Create a client certificate.
2. Export it to a file (*.pfx).
3. Add it to the project's "Certificates" folder.
4. Update the CertificateFileName variable in MainPage.xaml.cs with the certificate file name.

More info:
https://support.quovadisglobal.com/KB/a46/exporting-a-pfx-using-mmc.aspx
